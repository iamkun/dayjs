# dayjs

Is codenext feature #Real-Time Date

#### Usage

```javascript
const dayjs = require('dayjs')

let day = new dayjs.Dayjs()

day.local().date()

```

#### Compilation

document.getElementById("date"): Date

```javascript
var fullDateFormat = {
  YD: '1, 6-2', 
  M: '2, 13', 
  Y: '2, 13',
}

document.getElementById("date").innerHTML = dayjs.utc(fullDateFormat) //=> '2016-02-09 12:00:00'
```
